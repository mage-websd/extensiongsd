Thank you for downloading our extension. Please follow these instructions to proceed with the installation of your extension:

#1 Installation:

  1. Copy all files into your magento directory and turn Magento cache off.
  2. In case you have different then "default" default folder - rename folders app/design/frontend/default/default and skin/frontend/default/default.
  3. Navigate to System > Configuration > Templates-Master > Easy Flags and enable extension for any store you need.
  4. Configure Easy Flags module according to your needs.
  5. That's all. Navigate to your home page and check how your store looks.

If you have any problems with extension installation please contact us at templates-master.com/contacts/.
