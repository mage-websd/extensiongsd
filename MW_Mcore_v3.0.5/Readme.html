Mcore - License Activation Module

This is required extension that allows to manage and activate other MageWorld's extensions.

Installation

  1. Copy folders /app, /js, /skin to your magento location
  2. Refresh cache, logout and login again
  3. Go to Admin / Configuration / MageWorld / License Activation to manage and activate MageWorld's extensions

*Note:

  • The Mcore extension requires PHP SOAP enabled to activate the extensions online.
  • In case your sever does not enable PHP SOAP, please follow this article to fix:
    http://phpflow.com/php/class-'soapclient'-not-found/
  • Or you can activate the extensions offline using Offline Activation Key. See the License Activation user guide here:
    http://www.mage-world.com/wiki/index.php?title=License-activation
